  • n = value1 : 실행문1
  • n = value2 : 실행문2
  • default : if문의 else와 같음

5. for문

문법

1
2
3

for ( 초기식; 조건식; 증감식 ) {
    실행문;
}
  • 초기식
    • 초기식을 기준으로 조건과 증감식의 결과가 나온다.
    • 주로 int = 0;을 사용한다.
  • 조건식
    • 초기식의 반복 조건.
    • 조건식이 거짓이 되면 for문 종료
  • 증감식
    • 반복할 때 마다 초기식이 증감한다.
    • ++ 혹은 --을 쓴다
  • ex)
    1
2
3

    for (int i = 0; i < 10; i++) {
    printf("%d, ", i);
}

    결과 : 1, 2, 3, 4, 5, 6, 7, 8, 9,

중첩 for

int i;
int j;
for (i = 0; i <=5; i++ ) { // i가 5를 넘으면 for문 실행 x
    for (j = 0; j <=4; j++) {
        printf("i = %d, j = %d\n", i, j);
    }
}

i = 5, j = 4

6. Continue

  • 조건으로 다시 돌아간다.
  • ex)
    1
2
3
4
5
6
7
8
9

    int sum = 0;
for (int i=0; i<=6; i++) {
      
    if (i % 2 == 0) {
        continue; // 짝수일 경우 조건으로 돌아간다
    }
    sum += i; // sum = sum + i;
    printf("%d번째 sum값 = %d\n", i+1, sum);
}
    • 실행 순서
      • i = 0 » if : T » i++
      • i = 1 » if : F » sum += i; (sum=1), printf(), i++
      • i = 2 » if : T » i++
      • i = 3 » if : F » sum += i; (sum=4), printf(), i++
      • i = 4 » if : T » i++
      • i = 5 » if : F » sum += i; (sum=9), printf(), i++
      • i = 6 » if : T » i++
    • 결과
      • 2번째 sum값 = 1
      • 4번째 sum값 = 4
      • 6번째 sum값 = 9
